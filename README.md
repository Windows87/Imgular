## To use

First clone the repository:
```bash
# Clone this repository
git clone https://github.com/Windows87/imgular
# Go into repository
cd imgular
```

Start the Angular app:
```bash
# Go into Angular app
cd frontend
# Install dependencies
npm install
# Start
npm start
```

And start the server:
```bash
# Go into server
cd backend
# Install dependencies
npm install
# Start
npm start
```
